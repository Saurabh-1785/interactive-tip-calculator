## ✅ Features

- Enter bill amount
- Select preset or custom tip percentage
- Input number of people to split bill
- Live calculation of:
  - Tip amount
  - Total per person
  - Tip per person
- Reset button to clear inputs
- Fully responsive design (mobile + desktop)

## 🧱 Built With

- HTML5
- CSS3 (Flexbox/Grid)
- Vanilla JavaScript (ES6+)
